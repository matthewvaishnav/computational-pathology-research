# Documentation

This directory contains the GitHub Pages site for the Computational Pathology Research Framework.

## Site Structure

- `index.md` - Homepage
- `_config.yml` - Jekyll configuration
- `_layouts/` - Custom page layouts
- `assets/` - CSS, images, and other assets
- Other `.md` files - Documentation pages

## Local Development

To preview the site locally:

```bash
# Install Jekyll
gem install bundler jekyll

# Serve the site
cd docs
jekyll serve

# View at http://localhost:4000
```

## Deployment

The site is automatically deployed to GitHub Pages when changes are pushed to the `main` branch.

View the live site at: https://matthewvaishnav.github.io/computational-pathology-research/
